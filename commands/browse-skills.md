# Browse GitLab Skills

List all available GitLab automation skills.

## Available Skills

| Skill | Description |
|-------|-------------|
| gitlab-assistant | Hub skill - routes to specialized skills |
| gitlab-mr | Mr operations |
| gitlab-issue | Issue operations |
| gitlab-ci | Ci operations |
| gitlab-repo | Repo operations |
| gitlab-release | Release operations |
| gitlab-label | Label operations |
| gitlab-milestone | Milestone operations |
| gitlab-variable | Variable operations |


## Usage

To use a skill, simply describe what you want to do:

- "List all resources" - Routes to appropriate skill
- "Create a new item" - Routes to resource skill
- "Search for X" - Routes to search skill

## Getting Started

1. Run `/gitlab-assistant-setup` to configure authentication
2. Describe your task naturally
3. The assistant will route to the appropriate skill
